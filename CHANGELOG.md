# Changelog / История изменений

## [0.1.1] - 2025-11-03
### Added / Добавлено
- Added the “Remove audio track” option to segment dialogs to export silent video fragments.
- Updated user documentation to describe the audio removal workflow.
- Добавлена опция «Удалить аудиодорожку» в диалоги сегментов для экспорта видеоклипов без звука.
- Обновлена пользовательская документация, отражающая процесс удаления аудио.

## [0.1.0] - 2025-10-28
### Added / Добавлено
- Initial public release of Simple Video Slicer with PySide6 interface.
- Added the ability to build an executable file for different platforms.

- Добавлена возможность сборки исполняемого файла под разные платформы.
- Первая публичная версия Simple Video Slicer с интерфейсом на PySide6.
